# CSC102: The Foundation

This repository contains the HTML, CSS, and JavaScript assignment created for CSC102 - Introduction to Programming, Assignment 1.2: The Foundation.

## Overview

This project introduces the basics of building static web pages using HTML5, with CSS for styling and JavaScript to be added in later assignments. It includes two pages. The index.html page is a landing page featuring a page title, a header, and a fun school-appropriate meme image. The boards.html page contains my program objectives, pulled from the UAT University Catalog.

## Objectives Covered

Understand and code two working HTML pages. Translate technical requirements into working HTML. Use proper HTML5 structure and tags such as doctype, head, title, body, header, paragraph, and image. Practice clear code commenting throughout.
